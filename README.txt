  AI 生活管家 — 啟動說明

【第一次使用】安裝套件 + 建立資料庫
--------------------------------------
1. 開啟命令提示字元（cmd）或終端機
2. 切換到本專案資料夾：

   cd "C:\Users\Soon Yuan Chi\OneDrive\Desktop\比賽"

3. 安裝必要套件：

   pip install -r requirements.txt
   pip install streamlit

4. 建立資料庫（只需做一次）：

   python seed.py

================================================================

【啟動使用者前端（消費者介面）】
----------------------------------
執行：

   streamlit run app.py --server.port 8501

開啟瀏覽器，進入：

   http://localhost:8501

使用流程：
  1. 在文字框輸入生活需求，例如：
       「我家廚房水管漏水，想找人修」
       「幫我訂位，4個人，明天晚上」
       「家裡需要大掃除」
  2. 系統自動判斷服務類型並顯示對應表單
  3. 填寫表單內容後點「送出諮詢單」
  4. 畫面顯示諮詢單編號與媒合到的廠商

================================================================

【啟動廠商後台管理介面】
--------------------------
執行（另開一個終端機視窗）：

   streamlit run vendor_dashboard.py --server.port 8502

開啟瀏覽器，進入：

   http://localhost:8502

後台功能：
  - 頂部統計：總案件數 / 待處理數 / 未讀數 / 今日新增
  - 左側列表：可依「狀態」與「服務類型」篩選案件
  - 🔵 藍點 = 尚未讀取的新案件
  - 點「查看詳情」→ 右側顯示消費者填寫的完整內容
  - 狀態推進按鈕：待處理 → 已聯繫 → 已承接 → 已完成

================================================================

【同時開兩個介面的方式】
--------------------------
開兩個終端機視窗，分別執行：

  視窗 1：  streamlit run app.py --server.port 8501
  視窗 2：  streamlit run vendor_dashboard.py --server.port 8502

瀏覽器開兩個分頁：
  消費者：  http://localhost:8501
  廠商後台：http://localhost:8502

在 8501 填完表單送出後，重新整理 8502 即可看到新案件。
